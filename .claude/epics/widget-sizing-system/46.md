# Task: Etendre useBentoPrefs et useTeacherBento (persistence tailles)

## Description

Ajouter la persistence des tailles de widgets et du preset actif dans les composables de preferences existants, avec migration automatique des anciennes preferences.

## Acceptance Criteria

- [ ] `BentoPrefs` etendu avec `sizes: Record<string, WidgetSize>` et `preset?: string`
- [ ] `useTeacherBento` etendu avec le meme schema de tailles
- [ ] Migration automatique : anciennes prefs sans `sizes` → ajout de `sizes: {}` (tailles par defaut)
- [ ] `setWidgetSize(id, size)` : met a jour la taille d'un widget + efface le preset actif
- [ ] `getWidgetSize(id)` : retourne la taille sauvegardee ou la taille par defaut du registry
- [ ] `applyPreset(presetId)` : applique un preset complet (order, hidden, sizes)

## Technical Details

Fichiers : `src/renderer/src/composables/useBentoPrefs.ts`, `src/renderer/src/composables/useTeacherBento.ts`

Migration : detecter `!prefs.sizes` au chargement, initialiser avec `{}`. Les widgets sans taille sauvegardee utilisent `defaultSize` du registry.
