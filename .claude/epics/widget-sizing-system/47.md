# Task: Creer useWidgetGrid (logique responsive)

## Description

Creer un composable qui gere la logique de grille responsive : nombre de colonnes selon la largeur, clamping des tailles de widgets qui depassent la grille.

## Acceptance Criteria

- [ ] `useWidgetGrid(containerRef)` retourne `{ columns, gridClass, clampSize(size) }`
- [ ] Desktop (>1024px) : 4 colonnes
- [ ] Tablette (600-1024px) : 2 colonnes (widgets 4x1 → 2x1, widgets 2x2 → 2x2)
- [ ] Mobile (<600px) : 1 colonne (tous les widgets → 1x1 equivalent pleine largeur)
- [ ] `clampSize(size)` retourne la taille effective selon le nombre de colonnes actuel
- [ ] Utilise `ResizeObserver` sur le conteneur (pas `window.innerWidth`)

## Technical Details

Fichier : `src/renderer/src/composables/useWidgetGrid.ts` (nouveau)

Clamping :
- 4 cols : toutes tailles OK
- 2 cols : 4x1 → 2x1, reste inchange
- 1 col : tout → span 1
