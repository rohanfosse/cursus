# Task: Creer WidgetPicker visuel par categorie

## Description

Creer le nouveau picker de widgets qui remplace `BentoCustomizer.vue`. Panel lateral avec widgets organises par categorie, apercu visuel, toggle ajouter/retirer, et selecteur de taille.

## Acceptance Criteria

- [ ] Panel lateral (slide from right) avec les widgets par categorie
- [ ] Categories : Essentiel, Communication, Suivi, Productivite, Fun
- [ ] Chaque widget montre : icone, nom, description, tailles supportees
- [ ] Toggle "Ajouter/Retirer" avec animation
- [ ] Selecteur de taille (boutons radio) pour choisir avant d'ajouter
- [ ] Bouton "Reinitialiser" pour revenir aux defauts
- [ ] Fonctionne pour etudiant ET prof (filtre par `role`)

## Technical Details

Fichier : `src/renderer/src/components/dashboard/WidgetPicker.vue` (nouveau)

Remplace `BentoCustomizer.vue` pour les etudiants. Pour les profs, integre dans le mode edit de `TabAccueil.vue`.

Utilise le registry (`STUDENT_WIDGETS` ou `TEACHER_TILES`) pour lister les widgets disponibles.
