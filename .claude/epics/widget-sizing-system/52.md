# Task: Presets de layout + UI de selection

## Description

Creer les presets de layout pour etudiant et prof, et l'UI pour les selectionner (miniatures cliquables dans le widget picker ou le mode edit).

## Acceptance Criteria

- [ ] 3 presets etudiant : Compact (tout 1x1), Focus (projet 2x2 + essentiels), Equilibre (mix)
- [ ] 3 presets prof : Vue d'ensemble, Communication, Suivi
- [ ] Miniatures/previews des presets dans le picker
- [ ] Appliquer un preset remplace le layout actuel (avec confirmation modale)
- [ ] Apres application, l'utilisateur peut personnaliser librement
- [ ] Le preset actif est indique visuellement

## Technical Details

Fichier : `src/renderer/src/composables/useWidgetPresets.ts` (nouveau)

Chaque preset est un objet `BentoPrefs` statique :
```typescript
const STUDENT_PRESETS = {
  compact: { order: [...], hidden: [...], sizes: { ... }, preset: 'compact' },
  focus: { ... },
  balanced: { ... },
}
```
