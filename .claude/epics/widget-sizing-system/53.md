# Task: Resize interactif en mode edit

## Description

Ajouter la possibilite de redimensionner un widget en mode edit via un menu contextuel ou des boutons de taille dans le WidgetShell.

## Acceptance Criteria

- [ ] En mode edit, chaque widget affiche un bouton "resize" (ou menu au clic droit)
- [ ] Le menu montre les tailles supportees par le widget avec icones (grille miniature)
- [ ] La taille selectionnee est appliquee immediatement avec animation
- [ ] La nouvelle taille est sauvegardee dans les preferences
- [ ] Le preset actif est efface quand l'utilisateur resize manuellement

## Technical Details

Dans `WidgetShell.vue` : ajouter un bouton overlay en mode edit qui ouvre un petit popover avec les tailles disponibles (lues depuis le registry).

Animation : transition CSS `grid-column` + `grid-row` avec `transition: all 0.3s ease`.
