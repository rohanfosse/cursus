# Task: Composants FriseCalendar et FriseDotPopup

## Description

Creer deux nouveaux composants Vue reutilisables :
- `FriseCalendar.vue` : rendu generique de la frise horizontale (axe temporel ameliore + dots agreges)
- `FriseDotPopup.vue` : popup/dropdown au clic sur un dot agrege

## Acceptance Criteria

### FriseCalendar.vue
- [ ] Axe temporel avec separateurs de mois (lignes verticales)
- [ ] Fond alterne clair/fonce par mois
- [ ] Labels de mois toujours visibles sur l'axe
- [ ] Marqueur "aujourd'hui" : ligne verticale accent + label "Aujourd'hui"
- [ ] Dots normaux pour DayGroup avec 1 milestone
- [ ] Dots agreges (badge avec compteur) pour DayGroup avec 2+ milestones
- [ ] Taille du dot proportionnelle au nombre de milestones
- [ ] Drag horizontal, scroll molette, boutons zoom/navigation fonctionnels
- [ ] Props : dayGroups[], zoomLevel, offset, couleurs par projet
- [ ] Events : @milestone-click, @dot-click (pour popup)

### FriseDotPopup.vue
- [ ] Positionne sous le dot clique
- [ ] Liste chaque milestone : titre, type badge, projet couleur, deadline
- [ ] Chaque item cliquable (emit @select avec le milestone)
- [ ] Fermeture au clic exterieur ou Escape
- [ ] Slot ou prop pour contenu contextuel (responsable vs etudiant)
- [ ] aria-label pour accessibilite

## Technical Details

Fichiers :
- `src/renderer/src/components/frise/FriseCalendar.vue` (nouveau)
- `src/renderer/src/components/frise/FriseDotPopup.vue` (nouveau)

Extraire le rendu existant de TabFrise.vue (axe, dots, interactions) dans FriseCalendar. TabFrise devient un wrapper leger (toolbar + FriseCalendar).
