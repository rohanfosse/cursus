# Task: Logique de groupement par dots agreges (useFrise.ts)

## Description

Modifier le composable `useFrise.ts` pour grouper les milestones par date et produire des `DayGroup[]` au lieu d'empiler les dots verticalement. Un DayGroup avec 1 milestone produit un dot normal, 2+ produit un dot agrege avec compteur.

## Acceptance Criteria

- [ ] Nouveau type `DayGroup { date: string, milestones: FriseMilestone[], leftPct: number }`
- [ ] `teacherFrise` computed produit des `FrisePromo` contenant des `DayGroup[]` au lieu de `FriseMilestone[]`
- [ ] `studentFrise` computed produit egalement des `DayGroup[]`
- [ ] Un DayGroup avec 1 milestone = dot normal (meme comportement qu'avant)
- [ ] Un DayGroup avec 2+ milestones = dot agrege (compteur = milestones.length)
- [ ] La logique de zoom, drag, scroll existante n'est pas modifiee
- [ ] Les positions left% sont correctes pour chaque DayGroup

## Technical Details

Fichier : `src/renderer/src/composables/useFrise.ts`

Approche :
1. Ajouter le type `DayGroup` dans les types locaux ou `types/index.ts`
2. Dans les computed `teacherFrise`/`studentFrise`, grouper les milestones par `deadline.split('T')[0]`
3. Chaque groupe recoit le left% calcule a partir de la date
4. Exporter les DayGroup dans la valeur de retour du composable
