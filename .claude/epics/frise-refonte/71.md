# Task: Integration TabFrise + StudentTimelineModal

## Description

Integrer le composant FriseCalendar dans les deux vues existantes :
- TabFrise.vue : remplacer le rendu inline par FriseCalendar, garder la toolbar
- StudentTimelineModal.vue : remplacer la liste verticale par FriseCalendar + filtres + legende

## Acceptance Criteria

### TabFrise.vue
- [ ] Le rendu de la frise delegue a FriseCalendar
- [ ] La toolbar (zoom, navigation, bouton Aujourd'hui) reste dans TabFrise
- [ ] Clic milestone responsable → ouvre modale gestion devoir (meme comportement)
- [ ] Clic dot agrege → ouvre FriseDotPopup avec liste, puis clic item → modale devoir
- [ ] Pas de regression visuelle

### StudentTimelineModal.vue
- [ ] La liste verticale est remplacee par FriseCalendar horizontal
- [ ] Les filtres (A rendre / Rendus / Evenements) sont au-dessus de la frise
- [ ] La legende interactive (Rendu, Urgent, En retard, Evenement, A rendre) est sous la toolbar
- [ ] Les filtres filtrent les DayGroups affiches dans FriseCalendar
- [ ] Clic sur un dot etudiant → popup avec detail : statut, notes, feedback, countdown
- [ ] Le countdown se met a jour en temps reel (30s)

## Technical Details

Fichiers modifies :
- `src/renderer/src/components/dashboard/TabFrise.vue`
- `src/renderer/src/components/modals/StudentTimelineModal.vue`

Approche :
1. TabFrise : extraire le template de rendu frise, le remplacer par `<FriseCalendar :day-groups="..." ... />`
2. StudentTimelineModal : supprimer le template de liste verticale, ajouter FriseCalendar avec toolbar filtres/legende au-dessus
3. Le popup etudiant reutilise FriseDotPopup avec un slot/prop pour le contenu enrichi (notes, feedback)
