# Task: Tests backend (snapshot, zip, routes)

## Description

Couverture de tests vitest complete pour le service snapshot, le service zip et les routes REST du snapshot Lumen.

## Acceptance Criteria

### Service lumenSnapshot.js

- [ ] `parseGitHubUrl` : URLs valides, URLs invalides (http://, autre host, sans /owner/repo, avec trailing slash, avec .git)
- [ ] `buildSnapshot` : cas nominal (repo fictif via fetch mocke)

### Routes

- [ ] `POST /lumen/:id/snapshot` : 200 succes, 403 non-proprio, 404 cours inconnu, mapping des erreurs du service vers HTTP
- [ ] `POST /lumen/:id/snapshot/refresh` : 200 sans changement (`changed: false`), 200 avec changement (`changed: true`)
- [ ] `GET /lumen/:id/snapshot/download` : streaming zip complet, header `Content-Disposition`, 403 si etudiant d'une autre promo
