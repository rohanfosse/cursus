# Task: Integration LumenProjectPanel dans LumenReader (responsive)

## Description

Creer un composant wrapper `LumenProjectPanel.vue` qui assemble `LumenProjectTree` + `LumenProjectFileViewer` + bouton download, et l'integrer dans `LumenReader.vue` avec un layout responsive.

## Acceptance Criteria

- [ ] `LumenProjectPanel.vue` :
  - Props : `course: LumenCourse`
  - Charge l'arborescence via `store.fetchSnapshotTree(course.id)` au montage
  - Affiche un titre : "Projet d'exemple" + badge avec nombre de fichiers + taille
  - Layout interne : Tree a gauche (240px), FileViewer a droite (flex 1)
  - Bouton "Telecharger le projet (.zip)" en haut a droite, ouvre l'URL de download
  - Lien discret vers le repo GitHub original ("Voir sur GitHub" avec icone ExternalLink)
- [ ] Integration dans `LumenReader.vue` :
  - Affiche le panneau uniquement si `course.repo_snapshot_meta != null`
  - Desktop >= 1280px : en tiroir lateral droit (a cote du contenu de l'article, apres le TOC)
  - Desktop 1024-1280px : sous l'article, en section pleine largeur
  - Mobile : sous l'article, tree et viewer empiles verticalement, tree collapsible
  - Responsive transitions sans saccade
- [ ] Le panneau respecte le theme dark/light existant
- [ ] Le FileViewer est initialement vide (placeholder) jusqu'a la selection d'un fichier
- [ ] Scroll independant : le panneau ne perturbe pas le scroll principal de l'article ni de la progress bar

## Technical Details

- Nouveau composant : `src/renderer/src/components/lumen/LumenProjectPanel.vue`
- Modifier `src/renderer/src/components/lumen/LumenReader.vue` pour l'integrer dans la grille `reader-grid` existante (v2.23.0)
- Breakpoint desktop-large : ajouter une 3e colonne dans la grille (220px TOC + 1fr article + 380px project panel)
- Sur desktop 1024-1280px : garder la grille 2 colonnes (TOC + article), le panneau passe en pleine largeur sous l'article
- CSS : utiliser `@media (min-width: 1280px)` pour activer le layout 3 colonnes
- Le bouton download utilise `window.api.downloadLumenSnapshotUrl(courseId)` + `<a download>` ou `window.open`
