# Task: Service lumenZip — zip streaming depuis snapshot JSON

## Description

Creer `server/services/lumenZip.js` qui construit un zip streaming a partir du JSON de snapshot stocke en DB, reutilisable par la route de download.

## Acceptance Criteria

- [ ] Fonction `streamZipFromSnapshot(snapshot, writableStream, zipName)` qui pipe un zip dans `writableStream` (Node Writable, typiquement `res`)
- [ ] Le zip preserve les chemins relatifs des fichiers tels que dans le snapshot
- [ ] Contenu decode depuis base64 avant ecriture dans l'archive
- [ ] Streaming : pas de materialisation complete en memoire (utiliser `archiver` avec mode `zip`)
- [ ] Gere les erreurs d'ecriture (`error` event) avec un rejet de la promesse
- [ ] Ajoute un fichier `README-snapshot.txt` en racine du zip contenant `repo_url`, `commit_sha`, `fetched_at`

## Technical Details

- Ajouter `archiver` aux dependances npm (`npm install archiver`)
- Signature : `async function streamZipFromSnapshot(snapshotJson, res): Promise<void>`
- Format : `archiver('zip', { zlib: { level: 6 } })`
- Test avec un snapshot fixture (2-3 fichiers) qu'on unzippe en memoire pour verifier le contenu
