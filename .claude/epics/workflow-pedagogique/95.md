# Task: DepotsModal mode notation rapide (split view)

## Description

Ajouter un mode "Notation rapide" a la DepotsModal qui affiche une vue split : liste des etudiants a gauche, depot + notation a droite. L'etudiant selectionne montre son depot (apercu fichier/lien) et le formulaire de notation (A/B/C/D + commentaire).

## Acceptance Criteria
- [ ] Toggle "Notation rapide" dans le header de DepotsModal
- [ ] Vue split : liste etudiants (gauche 35%) + depot + notation (droite 65%)
- [ ] L'etudiant actif est mis en surbrillance dans la liste
- [ ] Apercu du depot : iframe pour fichiers PDF, lien cliquable pour URLs
- [ ] Formulaire : 4 boutons A/B/C/D (un seul actif), champ commentaire
- [ ] Indicateur progression : "12/25 notes" avec barre
- [ ] Filtre : Tous / Non notes / Notes (pour reprendre une session interrompue)
- [ ] Grade distribution en temps reel (pills A:x B:y C:z D:w)
- [ ] Le feedback bank existant est accessible
- [ ] Responsive : empilement vertical sous 768px

## Technical Details

- Fichier : `DepotsModal.vue` -- ajouter un mode alterne (pas un nouveau composant)
- Reutiliser `useTeacherGrading.ts` pour setNote/setFeedback
- Reutiliser le feedback bank existant dans DepotsModal
- State local : `batchMode: ref(false)`, `activeDepotIndex: ref(0)`
- Les 4 boutons A/B/C/D utilisent les classes `.grade-a`, `.grade-b`, etc. existantes

## Dependencies

Aucune -- independant de la publication programmee.

## Effort Estimate
- Size: L
- Hours: 4-6
