# Task: MultiPromoCard widget + composable useMultiPromo

## Description

Creer un composable `useMultiPromo.ts` qui charge les donnees de toutes les promos du prof, et un widget `MultiPromoCard.vue` qui affiche les metriques cle de chaque promo cote a cote.

## Acceptance Criteria
- [ ] `useMultiPromo.ts` : charge gantt + rendus pour chaque promo du prof
- [ ] Par promo : nombre de devoirs a venir, rendus en attente de note, progression globale
- [ ] `MultiPromoCard.vue` : 2 colonnes (ou cards) avec les metriques par promo
- [ ] Couleur de promo visible (badge ou bordure)
- [ ] Compteur "X a noter" bien visible (badge danger si > 0)
- [ ] Prochaines deadlines (top 3 par promo)
- [ ] Clic sur un devoir navigue vers la promo + ouvre la modale
- [ ] Dark mode
- [ ] Responsive : empilement vertical sous 768px

## Technical Details

- Composable : `src/renderer/src/composables/useMultiPromo.ts`
- Widget : `src/renderer/src/components/dashboard/MultiPromoCard.vue`
- API : reutiliser `window.api.getGanttData(promoId)` et `window.api.getAllRendus(promoId)`
- Navigation : `appStore.openChannel()` pour switch de promo + `modals.gestionDevoir = true`
- Les promos du prof sont deja disponibles via l'API existing (`getPromos` ou equivalent)

## Dependencies

Aucune -- independant des 2 autres phases.
