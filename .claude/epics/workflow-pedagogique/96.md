# Task: Navigation clavier + auto-save notation

## Description

Ajouter la navigation clavier complete au mode notation rapide et l'auto-save au changement d'etudiant. L'objectif est zero friction : le prof ne touche jamais la souris pendant une session de notation.

## Acceptance Criteria
- [ ] Fleche haut/bas : change d'etudiant dans la liste
- [ ] Touches A, B, C, D : applique la note directement (quand le focus n'est pas dans le textarea)
- [ ] Tab : deplace le focus vers le champ commentaire
- [ ] Enter (depuis commentaire) : sauvegarde + passe a l'etudiant suivant
- [ ] Escape : quitte le mode notation rapide
- [ ] Auto-save au changement d'etudiant (debounce 300ms)
- [ ] Feedback visuel : flash vert a la sauvegarde reussie
- [ ] Raccourcis documentes dans un hint en bas du panneau (comme GestionDevoirModal)
- [ ] Pas de double-save (guard pendant le save)

## Technical Details

- `@keydown` handler sur le conteneur du mode batch
- `useDebounceFn` de VueUse ou debounce manuel pour l'auto-save
- `useTeacherGrading.ts` : `saveGrade()` deja existant
- State : `pendingNoteValue`, `pendingFeedbackValue` du composable
- Flash animation : class CSS `.grade-saved` avec transition opacity

## Dependencies

- Tache 5 (split view)
