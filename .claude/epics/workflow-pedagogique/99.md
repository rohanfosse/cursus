# Task: Integration dashboard + navigation cross-promo

## Description

Integrer le widget MultiPromoCard dans le dashboard enseignant. Wirer la navigation cross-promo : clic sur un devoir dans la carte multi-promo switch automatiquement la promo active et ouvre la modale.

## Acceptance Criteria
- [ ] MultiPromoCard affiche en haut du dashboard teacher (avant les widgets existants)
- [ ] Visible uniquement si le prof a 2+ promos
- [ ] Clic sur un devoir : `appStore.activePromoId` switch, puis ouvre GestionDevoirModal
- [ ] Transition fluide (pas de rechargement complet de page)
- [ ] Le widget se rafraichit quand on revient au dashboard

## Technical Details

- Dashboard : `src/renderer/src/views/DashboardView.vue` ou composant equivalent
- Le switch de promo utilise `appStore.openChannel(channelId, promoId, channelName)`
- Apres le switch, ouvrir la modale via `appStore.currentTravailId = devoirId; modals.gestionDevoir = true`

- Tache 8 (MultiPromoCard)
