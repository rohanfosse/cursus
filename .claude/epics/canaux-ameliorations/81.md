# Task: Gestion membres ChannelMembersPanel

## Description

Enrichir le composant `ChannelMembersPanel` pour permettre la gestion des membres apres la creation d'un canal :

- Bouton "+" avec recherche d'etudiants de la promo pour ajouter un membre
- Bouton "x" pour retirer un membre (visible uniquement pour le responsable)
- Appel a `updateChannelMembers` existant pour persister les modifications
- Afficher `ROLE_LABELS` sur chaque membre (ex: "Responsable", "Etudiant")
- Toasts de confirmation pour chaque action (ajout/retrait)
- Le panneau n'est visible que pour le responsable et uniquement sur les canaux prives

## Acceptance Criteria

- [ ] Le bouton "+" ouvre une recherche d'etudiants de la promo
- [ ] Le bouton "x" permet de retirer un membre du canal
- [ ] Les modifications sont persistees via `updateChannelMembers`
- [ ] Chaque membre affiche son role via `ROLE_LABELS`
- [ ] Un toast confirme l'ajout ou le retrait d'un membre
- [ ] Le panneau n'est visible que pour le responsable sur les canaux prives

## Technical Details

- Modifier : `ChannelMembersPanel.vue` (boutons +/x, recherche, roles)
- Utiliser : `updateChannelMembers` existant dans le store/IPC
- Utiliser : `ROLE_LABELS` pour l'affichage des roles
- Utiliser : `showToast()` pour les confirmations
