# Task: Header canal enrichi + toasts + badges mentions

## Description

Ameliorer l'experience utilisateur des canaux sur trois points :

1. **Header enrichi** : afficher la description du canal, le compteur de membres et un badge indiquant le type (public/prive) dans le header de `MessagesView`
2. **Toasts** : ajouter des notifications toast sur les actions create/rename/delete de canaux
3. **Badge mentions** : creer un badge visuellement distinct pour les mentions (@) par rapport au badge unread existant dans la sidebar

## Acceptance Criteria

- [ ] Le header de MessagesView affiche la description du canal
- [ ] Le header affiche le compteur de membres
- [ ] Le header affiche un badge indiquant le type du canal (public/prive)
- [ ] Un toast est affiche lors de la creation d'un canal
- [ ] Un toast est affiche lors du renommage d'un canal
- [ ] Un toast est affiche lors de la suppression d'un canal
- [ ] Le badge mention (@) est visuellement distinct du badge unread dans la sidebar

## Technical Details

- Modifier : `MessagesView.vue` (header enrichi avec description, compteur, badge type)
- Modifier : `useSidebarActions.ts` (toasts sur create/rename/delete)
- Modifier : `Sidebar.vue` ou composant de canal (badge mention distinct)
- Les donnees sont deja disponibles dans `appStore` (description, type)
